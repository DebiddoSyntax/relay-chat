RelayChat 
**Live Demo:** [https://reelaychat.vercel.app/](https://reelaychat.vercel.app/)

RelayChat is a real-time messaging application built for seamless everyday communication. It supports private chats, group conversations, and live interaction through a clean and responsive interface.

Users can chat one-on-one or in groups, with messages delivered instantly using WebSocket technology. For deeper interaction, RelayChat also includes web-based calling powered by WebRTC, enabling real-time voice and video calls directly from desktop browsers.

A standout feature is Sydney, a personalized AI assistant integrated into the chat experience. Sydney can assist with conversations, provide quick responses, and make interactions more engaging and productive.

Key features include:

* Private one-on-one messaging
* Group chats
* Real-time messaging with WebSockets
* Voice and video calls on desktop using WebRTC
* Personalized AI assistant (Sydney)
* Secure authentication system

RelayChat is built with a modern full-stack architecture focused on performance, scalability, and a smooth user experience across devices.
